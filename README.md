# Hyeonseok's Portfolio & Blog

개발자 주현석의 개인 포트폴리오와 개발 블로그를 통합하여 관리할 수 있는 개인화 플랫폼 MVP입니다. 
Django와 HTMX, Alpine.js를 조합하여 화면 새로고침 없는 부드러운 사용자 경험(UX)과 깔끔하고 미니멀한 UI를 목표로 개발되었습니다.

---

## 📌 주요 기획 및 기능

### 1. 포트폴리오 (Portfolio)
* **프로필 및 기술 스택**: 이름, 한 줄 소개, 기술 스택 뱃지 등을 포함한 프로필 영역을 제공합니다.
* **이력 관리**: 교육 이수, 프로젝트/경력, 수상 내역, 자격증, 연구 등의 항목을 섹션별로 분리하여 깔끔하게 레이아웃을 구성했습니다.
* **관리자 편집 모드**: 관리자 계정으로 로그인 시 화면에서 실시간으로 각 항목을 추가, 수정, 삭제할 수 있습니다.
* **PDF 다운로드**: 작성된 이력을 언제든지 PDF 포맷으로 저장할 수 있습니다.

### 2. 블로그 & 콘텐츠 관리 (Blog)
* **마크다운 기반 에디터**: 게시글 작성 및 수정 시 왼쪽에는 마크다운 작성란, 오른쪽에는 실시간 미리보기(Live Preview)가 작동하는 직관적인 에디터를 구현했습니다.
* **이미지 드래그 앤 드롭 & 클립보드 업로드**: 이미지를 끌어다 놓거나 클립보드에서 붙여넣기(Ctrl+V)하면 자동으로 서버 업로드 후 마크다운 태그를 삽입합니다.
* **시리즈(Series)화**: 연관된 글들을 모아서 연재할 수 있도록 시리즈 관리 기능을 탑재하고, 상세 페이지 하단에 이전 편/다음 편 네비게이션을 제공합니다.
* **실시간 검색 및 필터링**: 제목 및 본문 전체 텍스트 검색을 지원하며, 시리즈별 필터링 기능이 새로고침 없이 동작합니다.

### 3. 회원 기능 및 커뮤니티
* **이메일 로그인**: 번거로운 아이디 생성 대신 이메일과 비밀번호 기반의 깔끔한 가입 및 로그인 시스템을 적용했습니다.
* **커스텀 프로필**: 마이페이지에서 본인의 정보 수정 및 커스텀 프로필 사진 업로드가 가능하며, 프로필 사진이 없을 시 닉네임을 기반으로 하는 고유한 로봇 아바타가 자동 매핑됩니다.
* **댓글 & 대댓글**: 게시글 상세 화면에서 부드러운 답글(대댓글) 쓰기가 가능하며, 작성자가 삭제하더라도 맥락을 유지하기 위해 소프트 삭제(Soft Delete) 형태로 처리됩니다.
* **인터랙티브 이모지 반응**: 댓글마다 👍, ❤️, 😂 등 다양한 이모지 반응을 남길 수 있으며, 게시물 좋아요 기능도 함께 작동합니다.

### 4. 통합 관리자 대시보드 (Admin Dashboard)
* Django 순정 어드민 페이지 외에도, 한 화면에서 유저 활성화/비활성화 제어, 게시글 공개/비공개 전환, 시리즈 삭제 등을 직관적으로 처리할 수 있는 커스텀 관리자 콘솔을 구축했습니다.

## 🛠 기술 스택

### Backend
* **Django**: MVC 프레임워크 및 ORM 기반 데이터 모델링
* **Pillow**: 사용자 프로필 및 블로그 업로드 이미지 처리
* **django-environ**: 환경변수(.env) 설정 및 보안 관리
* **Python-Markdown**: 서버사이드 마크다운 렌더링 파이프라인

### Frontend
* **HTMX**: Ajax 요청을 HTML 속성으로 직접 처리하여 SPA와 같은 부드러운 페이지 전환 및 검색 구현
* **Alpine.js**: 다크모드 설정 보존(LocalStorage), 모바일 사이드바 토글, 인라인 답글 입력창 토글 등 가벼운 프론트엔드 상태 제어
* **Vanilla CSS**: CSS Custom Properties 기반의 다크/라이트 테마 전환 및 모던 프리미엄 UI 스타일링
* **Highlight.js**: 코드 블럭 신택스 하이라이팅

## ⚙️ 실행 방법 (Local Development)

1. **저장소 클론 및 가상환경 구성**
   ```bash
   git clone https://github.com/juhyeonseok/portpolio.git
   cd portpolio
   python -m venv venv
   ```

2. **가상환경 활성화 (Windows 기준)**
   ```bash
   venv\Scripts\activate
   ```

3. **의존성 라이브러리 설치**
   ```bash
   pip install -r requirements.txt
   ```

4. **환경 변수 파일 생성**
   루트 경로에 `.env` 파일을 생성하고 아래의 정보들을 알맞게 기입합니다.
   ```env
   SECRET_KEY=your_secret_key_here
   DEBUG=True
   ALLOWED_HOSTS=127.0.0.1,localhost
   ```

5. **마이그레이션 진행 및 슈퍼유저 생성**
   ```bash
   python manage.py migrate
   python manage.py createsuperuser
   ```

6. **로컬 서버 구동**
   ```bash
   python manage.py runServer
   ```
